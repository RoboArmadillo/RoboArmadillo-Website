Servos

You have a maximum of 8 servos that you can use. However the 6 AA batteries on the kit can really only supply enough current for 2 to work at a time. Therefore you will have to attach a seperate battery to the servo board to power all 8.The 8 servos are contained within a list called servos (therefore accessed using R.servos[0],R.servos[1] etc). Each of the elements of this list have an element called "angle" (if you have mastered using motors this bit should be self-explainatory). In order to set the angle that you want the servo to go to you just change this "angle" property. This property can be set, at any integer value betwen 0-180.

Setting Digital output Example

R.servos[0].angle = 40

This would set the first servo to the 40 degrees position.
