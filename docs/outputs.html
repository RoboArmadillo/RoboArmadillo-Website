Outputs

RoPyPiKokiDuino handles Outputs in a very similar way to how it handles Inputs except that you are allowed to set them as opposed to just reading them! You have 16 digital outputs at your disposal. They are contained within the list called outputs (there accessed using R.outputs[0],R.outputs[0] etc). Each element only has one attribute "d" standing for digital. This is a "boolean" and thus can be set to either "True" or "False". If you set the output "True" it goes to 5 Volts, if you set it "False" it goes to 0 Volts.

Setting Digital output Example

R.outputs[0].d = True

This sets digital output 0 to 5V

R.outputs[0].d = False

This would set digital output 0 to 0V
