Distance orderer

def distance_orderer(listname): #where listname is the list of markers to be ordered

listname=sorted(listname, key=lambda Marker:Marker.dist)

return listname

You would use this in the following way:

markers = R.see((1080,1024),True,1)

markers = distance_orderer(markers)

The markers list would now be in order based upon how far the marker is from the camera (closest first).
