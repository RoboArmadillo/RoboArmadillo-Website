Inputs

You have 16 analogue and 16 digital inputs at your disposal. They are contained in a list called "inputs" (In exactly the same way as the motors). Now it gets slightly complicated. Each element of the list has TWO properties. One is called "a" and the other is "d" where "a" stands for analogue and "d" for digital. If I read the value of "a" property of element 2 in the list "inputs" (R.inputs[1].a) this will give me a voltage between 0-5V as a float (a number with decimal point in it) reading from pin A1 (the 2nd analogue pin). BUT if I were to read from R.inputs[1].d it would give return a "Boolean"(True or False) from pin 31 (the 2nd digital pin-for more details on which pins are which please see the Arduino Mega diagram on the Useful Extras page .)

Read Digital Inputs Example

bump_switch = R.inputs[0].d

print bump_switch

>>True

This sets the variable bump_switch to equal digital input 0, and then returns True or False.

Read Analog Input Example

light_sensor = R.inputs[0].a

print lightsensor

This would then return a float from 0.00-5.00

Warning

When plugging in your sensors, as said before R.input[0].a is not the same pin as R.input[0].d
