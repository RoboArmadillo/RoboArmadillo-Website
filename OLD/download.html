So you want to make your very own RoPiPyKokiDuino kit? You have come to the right place! There are a number of steps which you must take to make yourself a working kit. There are two sides to it however. Firstly there is the actual construction the kit itself and then there is loading the software onto the Pi and the Arduino. By following the easy to use step by step guide below you will have a kit ready and working in no time at all!

Section 1 - Preparing the Pi

In a terminal type the following:

sudo apt-get update

sudo apt-get upgrade sudo apt-get install libopencv-dev libyaml-dev freeglut3-dev scons python-picamera python-rpi.gpio python-serial python-opencv

Download and extract Pikoki from here #put link here

In the Pikoki folder:

sudo scons

Note that if you get an error about permissions, in the Pikoki folder:

sudo chmod 755 create-pkg-config

copy placeYouExtracted/lib/libkoki.so to /usr/lib

In the PiPyKoki folder:

sudo python setup.py install
