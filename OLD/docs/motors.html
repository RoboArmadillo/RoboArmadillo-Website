Motors

You have four PWM motors available for use. This means you can vary their speeds and direction. These four motors are contained in a list called "motors" which is part of the "robot" object which you made an instance of earlier called "R". Lists in virtually every programming language start from element 0 (for the motors list element 0 would be called R.motors[0]. Each of the motors has an "attribute" called speed. When you set this it changes how fast the motor turns. You can set it between -100 and 100 percent speed. -100 would be backwards at full speed and 100 would be forwards at full speed.

Therefore if you wanted to set motor 3 to 57 percent you would use:

R.motors[2].speed = 57

or perhaps I want it to go backwards at 34%:

R.motors[2].speed = -34
